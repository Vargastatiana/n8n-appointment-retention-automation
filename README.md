# 🚀 Sistema de Agendamiento y Retención Automática

Esta clínica de estética perdía ingresos por un seguimiento manual ineficiente. Para solucionarlo, creé este ecosistema automatizado en n8n que se dispara en tiempo real con cada nueva cita. El flujo procesa los datos, realiza el formateo de campos necesario y se conecta con la API de Google Calendar para agendar el servicio de forma inmediata.

---

### 🛠️ Stack Tecnológico
* **Core:** n8n
* **Database:** Google Sheets API
* **Calendar:** Google Calendar API
* **Email:** Gmail API
